

# Arsenal51 Orbiter Android App
A native app for Android for discovering and controlling your WLED-Arsenal51 devices easily!  

Forked from original project by Moustachauve which can be found [here](https://github.com/Moustachauve/WLED-Android)

### Features
- Automatic device detection (mDNS)
- All lights are accessible from one list
- Custom names
- Opens control UI immediately if connected to Orbiter-AP
- Hide or delete devices
- Tablet support for a better experience on wider screens
